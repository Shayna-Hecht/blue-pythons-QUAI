About QUAI

QUAI is an AI model which takes in a question and rephrases it.

How we made it

How we made QUAI

We used a model called GPT-2 provided by AItextgen which was trained on many sources, including all of Wikipedia, to gain an understanding of the English language. We found a dataset on Kaggle to train the model more for our specific purpose. The dataset consisted of repetitive questions from Quora which were paired together. The AI model would take in a question and try to predict the next question one word at a time. By doing this many times with many questions the model learned how to rephrase questions.

How it works

QUAI uses a neural network, which is a function class. A function class is just a bunch of functions. The neural network figures out what function to use by adjusting the function it is using during training. It adjusts this so that it can map the input closer to the target output in the training data than before. What QUAI does is convert the inputs into vectors, or tables of numbers based on certain attributes. There are vectors for each word and for the contexts of words. Then these vectors are run through the neural network to come up with output vectors, which are then converted into words and outputted.

What is QUAI for?

QUAI takes in a question and rephrases it. In so doing, QUAI solves the problem of overfitting because it allows students to learn the concepts a question is meant to assess instead of the specific wording of any question. Essentially, QUAI helps you understand rather than just memorize a concept.
